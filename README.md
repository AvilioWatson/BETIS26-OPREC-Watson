### 1. Forking Repository

Jangan melakukan perubahan langsung pada repositori pusat. Salin rute ini ke akun Anda:

1. Klik tombol **Fork** di pojok kanan atas halaman GitHub ini.
2. Pilih akun GitHub pribadi Anda sebagai tujuan fork.
3. Setelah proses selesai, **Clone** repositori hasil fork tersebut ke mesin lokal Anda.

### 2. Instalasi Perlengkapan

Proyek ini menggunakan dua teknologi utama yang harus dijalankan secara bersamaan.
(PENTING: Pastikan anda menggunakan `pnpm` untuk menghindari dependency conflict. Aktifkan dengan `npm install -g pnpm`)

```bash
pnpm install
pnpm dev
```
